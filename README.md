# Zipoly 完整介绍
Zipoly 是一款**本地离线3D模型轻量化、格式转换桌面工具**，数字孪生/Web3D从业者自研，主打保密项目离线处理，适配Three.js、数字大屏、园区/机场孪生场景。
## 一、核心优势（区别在线工具）
1. **纯本地离线**
基于Tauri开发Windows客户端，所有模型运算在本机完成，不上传任何文件到服务器；军工、工业涉密模型可用，授权仅校验本地硬件ID，全程断网可用。
2. **双几何压缩引擎**
内置行业标准两种压缩方案，一键切换：
- Draco（KHR_draco_mesh_compression）：通用轻量化，兼容性强
- Meshopt（gltfpack）：压缩率更高，支持KTX2纹理一体化压缩
几何整体压缩率 **70%~90%**，40MB模型可压至7MB左右，肉眼无明显失真。
3. **一体化纹理优化**
集成Basis Universal KTX2压缩（ETC1S/UASTC），自动缩放超大贴图、批量转WebP/压缩PNG/JPG，解决大贴图占体积痛点。

## 二、四大核心功能
### 1. 模型压缩优化（核心）
- 单文件/整文件夹批量递归处理，多线程并发，支持暂停、日志记录
- 内置预设：Web网页、移动端、工业高精度、电商产品
- 精细可调：顶点量化精度、纹理分辨率、法线压缩、减面阈值
- 输出标准GLB/glTF，开箱适配Three.js、Unity、虚幻

### 2. 多格式互转
支持7种主流3D格式双向转换：
`FBX / OBJ / STL / DAE / PLY ↔ GLB / glTF`
自动识别OBJ配套`.mtl`材质与贴图，极少丢材质，解决传统转换工具材质丢失问题。

### 3. 内置3D实时查看器
- 直接加载所有支持格式，多模型同屏预览
- 自定义光源、日照光影、网格信息（顶点/三角面/贴图数量）
- 压缩/转换后即时校验模型、贴图是否正常

### 4. 操作日志
本地保存全部处理参数，可一键复用历史压缩配置，批量统一规格。

## 三、使用场景
1. 数字孪生：园区、机场、工厂超大场景批量轻量化
2. Web3D网页展示：产品3D落地页、在线可视化
3. 涉密工业/医疗/军工模型：禁止上传云端，必须本地处理
4. 建模管线：Blender/3ds Max导出FBX/OBJ批量转Web可用GLB

## 四、渠道
- 桌面客户端官网：`zipoly.netlify.app`（Windows安装包约5MB）
- 配套轻量在线预览版：zipoly-web（仅预览，无离线压缩能力）

## 六、典型工作流（数字孪生标准流程）
1. 3D软件导出FBX/OBJ整套模型贴图
2. Zipoly「格式转换」批量转GLB，自动修复材质
3. 切换「压缩优化」，选用Web预设批量轻量化
4. 内置查看器校验贴图、结构无误
5. 输出GLB直接接入Three.js大屏项目
